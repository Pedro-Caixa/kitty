RegisterUtils​

</>
Kitty.RegisterUtils(utilsFolder: Instance) → ()

Registers utility modules from a folder into Kitty.Utils.

If a utility name matches an existing utility, the custom one replaces it.

RegisterPackages​

</>
Kitty.RegisterPackages(packagesFolder: Instance) → ()

Registers package modules from a folder into Kitty root.

Each ModuleScript is exposed as Kitty.. If a name matches an existing Kitty member, the custom package replaces it.
